# Local Network Port Scanner

## Project Overview
This project is a Python-based cybersecurity tool that scans a system for open network ports.
Open ports can represent security risks if not properly secured.

## Tools Used
- Python 3
- Socket library

## Installation
1. Install Python 3
2. Clone the repository
3. Run `python port_scanner.py`

## Usage
The tool scans common service ports and reports whether they are open or closed.

## Security Relevance
Port scanning helps administrators identify exposed services that attackers may target.

## Challenges
- Managing scan timeouts
- Selecting relevant ports
